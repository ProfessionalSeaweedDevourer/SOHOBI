# SOHOBI 프로덕트 백로그 — 에이전트 플러그인 중심 50선

## Context

현재 SOHOBI 운영 중인 인프라:
- **에이전트 도메인:** admin, finance, legal, location, chat
- **플러그인:** SeoulCommercialPlugin, FinanceSimulationPlugin, LegalSearchPlugin (Azure AI Search), GovSupportPlugin, AdminProcedurePlugin
- **체크리스트:** 8개 항목 (biz_type, location, capital, biz_reg, permit, labor, finance_sim, lease)
- **로드맵 투표 중:** tax_guide, hr_guide, menu_pricing, safety_checklist, delivery_pos, inventory, crm, revenue_dashboard

**점수 = 이점(1–5) × 2 − 난이도(1–5)**

**작업 파일:**

- `integrated_PARK/roadmap_router.py` — ROADMAP_FEATURES에서 tax_guide 상태 업데이트
**완료 기준:** `/api/v1/query`에 세무·지원금 질문 시 도메인 `admin`으로 라우팅, 구체적 답변 반환
