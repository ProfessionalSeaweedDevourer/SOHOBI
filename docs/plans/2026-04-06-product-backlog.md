# SOHOBI 프로덕트 백로그 v1.0

**작성일:** 2026-04-06
**작성자:** PARK 브랜치 (Claude Code 보조)
**대상 독자:** SOHOBI 팀 전원

---

## 이 문서의 목적

SOHOBI에 추가할 수 있는 기능 50가지를 선별하고, 구현 우선순위와 팀 작업 배분을 정리한 로드맵 문서입니다.
각 기능은 **"사용자에게 얼마나 가치 있는가"** 와 **"얼마나 빠르게 만들 수 있는가"** 두 축으로 점수화했습니다.

> **점수 공식:** 이점(1–5) × 2 − 난이도(1–5)
> 점수가 높을수록 빠르게 큰 효과를 낼 수 있는 기능입니다.

---

## 현재 SOHOBI가 제공하는 것

현재 서비스는 5개 전문 에이전트를 운영 중입니다:

| 에이전트 | 역할 |
|---------|------|
| **행정(admin)** | 영업신고, 위생교육, 사업자등록 안내 |
| **재무(finance)** | 손익 시뮬레이션, 몬테카를로 수익 분석 |
| **법무(legal)** | 임대차·권리금·상가임대차보호법 상담 |
| **상권(location)** | 서울 2024년 4분기 상권 데이터 분석 |
| **대화(chat)** | 일반 서비스 안내 |

사용자 설문 기반 로드맵 투표 기능도 운영 중이며, 현재 8개 후보(세무 안내, HR/노무, 메뉴 원가, 위생 점검, 배달 수수료, 재고관리, CRM, 매출 대시보드)가 투표 중입니다.

## Phase 1 — 지금 바로 시작 가능한 기능 (10개)

> **공통 조건:** 기존 에이전트 코드에 플러그인 클래스만 추가하면 됩니다.
> 외부 API 계약 없이 즉시 착수 가능합니다.

### 1위 — 세무 신고 가이드 에이전트 ★ (점수 8)

**사용자 가치:**
부가세 신고를 처음 해보는 자영업자는 "언제, 어디서, 어떤 서류로" 신고하는지 몰라 세무사를 찾아갑니다. SOHOBI가 이 질문에 즉시 답해줄 수 있습니다.

**커버 범위:** 부가세 신고 시기, 간이과세자 vs 일반과세자 구분, 종합소득세 신고 방법, 홈택스 사용법
**구현 방향:** `admin_agent.py`에 `TaxGuidePlugin` 추가. 기존 `LegalSearchPlugin`(법령 검색)을 재사용.
**로드맵 투표 현황:** `tax_guide` 항목 투표 중 — 사용자 수요 검증됨.

---

### 2위 — 정부지원금 자동 매칭 에이전트 ★ (점수 8)

**사용자 가치:**
"나 같은 사람도 정부 지원금 받을 수 있나요?" 가 가장 많이 오는 질문 유형입니다. 업종·나이·소재지를 입력하면 수혜 가능한 지원사업 목록과 신청 방법을 알려줍니다.

**설계 원칙:** 외부 DB가 없어도 LLM 내재 지식으로 즉시 동작합니다. DB 연동은 추후 고도화 옵션.
**커버 범위:** 소상공인 정책자금, 청년 창업패키지, 노란우산공제, 지역 신보 보증, 고용지원금
**구현 방향:** 기존 `GovSupportPlugin` 확장 + 프로파일링 로직 추가.

---

### 3위 — HR/노무 에이전트 (점수 6)

**사용자 가치:**
직원을 처음 고용하는 사장님은 "최저임금이 얼마인지", "근로계약서를 어떻게 쓰는지", "해고할 때 절차가 뭔지" 를 모릅니다.

**커버 범위:** 최저임금, 근로계약서 작성법, 4대보험 가입 의무, 해고예고제도, 퇴직금 계산
**구현 방향:** `legal_agent.py` 패턴을 그대로 복제해 `hr_agent.py` 신규 생성. 오케스트레이터에 `hr` 도메인 등록.
**로드맵 투표 현황:** `hr_guide` 투표 중.

---

### 4위 — 메뉴 원가 계산 에이전트 (점수 6)

**사용자 가치:**
"아메리카노를 4,500원에 팔면 남는 게 있나요?" — 식재료비, 인건비, 임대료를 종합한 원가율 계산과 적정 판매가 제안.

**커버 범위:** 식재료 원가율 계산, 목표 마진 기반 판매가 역산, 메뉴별 공헌이익 비교
**구현 방향:** `finance_agent.py`에 `MenuCostPlugin` 추가.
**로드맵 투표 현황:** `menu_pricing` 투표 중.

---

### 5위 — 위생/안전 점검 에이전트 (점수 6)

**사용자 가치:**
음식점 위생 점검에서 어떤 항목을 보는지, HACCP 인증이 뭔지, 소방 시설 기준이 어떻게 되는지 대화형으로 확인합니다.

**구현 방향:** `AdminProcedurePlugin`에 `SanitationCheckPlugin` 추가. 체크리스트 형태로 제공.
**로드맵 투표 현황:** `safety_checklist` 투표 중.

---

### 6위 — 계절성 매출 예측 에이전트 (점수 7)

**사용자 가치:**
"여름에 빙수집 하면 겨울엔 어떻게 버티나요?" — 기존 몬테카를로 시뮬레이션에 계절 지수를 결합해 분기별 현금흐름 차트 제공.

**기술적 강점:** 서울 상권 DB에 이미 분기별 매출 데이터가 있어 계절 지수 추출 가능. SOHOBI만의 차별화 포인트.
**구현 방향:** `finance_agent.py` + 상권 DB 쿼리 결합.

---

### 7위 — 계약서 분석 에이전트 (점수 7)

**사용자 가치:**
임대차 계약서에 어떤 독소조항이 있는지 혼자 파악하기 어렵습니다. 계약서 텍스트를 붙여넣으면 위험 조항을 짚어주고 협상 포인트를 제안합니다.

**구현 방향:** `legal_agent.py`에 `ContractAnalysisPlugin` 추가. 텍스트 붙여넣기로 즉시 사용 가능.
**Phase 3에서:** PDF 업로드 지원으로 확장 예정.

---

### 8위 — 배달앱 수수료 최적화 에이전트 (점수 5)

**사용자 가치:**
쿠팡이츠·배민·요기요 수수료 구조가 제각각입니다. 내 메뉴 단가와 주문량 기준으로 플랫폼별 실질 마진을 계산합니다.

**구현 방향:** `finance_agent.py`에 `DeliveryFeePlugin` 추가. 수수료 구조 데이터 내장.
**로드맵 투표 현황:** `delivery_pos` 투표 중.

---

### 9위 — 상권 트렌드 모니터링 에이전트 (점수 5)

**사용자 가치:**
"홍대 카페 요즘 많이 폐업하나요?" — 특정 지역·업종의 분기별 개폐업률 추이와 포화도를 차트로 보여줍니다.

**구현 방향:** `location_agent.py`에 `TrendMonitorPlugin` 추가. 기존 상권 DB를 시계열 분석으로 활용.

---

### 10위 — 폐업/양도 절차 에이전트 (점수 4)

**사용자 가치:**
창업 못지않게 폐업·양도 절차를 모르는 경우가 많습니다. 권리금 회수 방법, 세무 정리, 고용보험 신고까지 단계별 안내.

**구현 방향:** `admin_agent.py` 확장.

## 기술 노트 — 다음 세션 인수인계

**즉시 활용 가능한 기존 인프라:**
- `LegalSearchPlugin`: Azure AI Search 기반 법령 RAG — 세무 가이드에도 재사용 가능
- `FinanceSimulationPlugin`: 몬테카를로 10,000회 — 계절 지수만 추가하면 계절성 예측으로 확장 가능
- `GovSupportPlugin`: 정부지원금 텍스트 생성 기반 — LLM fallback으로 즉시 강화 가능
- 체크리스트 자동 매칭(`_auto_check()`): 신규 에이전트 추가 시 키워드만 등록하면 자동 체크 연동

**주의 사항:**
- 오케스트레이터(`orchestrator.py`)에 신규 도메인 등록 시 Sign-off 루브릭도 함께 추가
- 로드맵 페이지(`roadmap_router.py`)의 `ROADMAP_FEATURES`는 Phase 1 기능 완성 시 상태 업데이트 필요
- `.env` 파일에 Azure API 키 있음 — 절대 커밋 금지
