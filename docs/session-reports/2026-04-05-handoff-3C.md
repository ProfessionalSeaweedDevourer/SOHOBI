### 동작 방식
- `MyReport.jsx`가 마운트 시 `localStorage.getItem("sohobi_session_id")`로 세션 ID를 읽어 `GET /api/report/{id}` 호출
- 세션 없으면 "AI 에이전트와 대화를 시작해 보세요" 안내, 에러 시 에러 메시지 표시
- `report_router.py`는 asyncio.gather로 3개 컨테이너를 병렬 집계; Cosmos DB 미설정 시 각 모듈의 인메모리 폴백 사용
