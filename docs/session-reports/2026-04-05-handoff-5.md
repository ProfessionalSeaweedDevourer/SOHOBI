### 작업 2: 보안 취약점 수정

아래 항목을 심각도 순으로 수정한다.

#### 2-1. IDOR 취약점 — CRITICAL

비로그인 사용자가 임의의 `session_id`를 URL에 넣어 다른 사용자 데이터 열람 가능.

| 엔드포인트 | 파일 | 문제 |
|-----------|------|------|
| GET /api/checklist/{session_id} | `integrated_PARK/checklist_router.py:21` | 소유권 검증 없음 |
| PATCH /api/checklist/{session_id} | `integrated_PARK/checklist_router.py:37` | 소유권 검증 없음 |
| GET /api/report/{session_id} | `integrated_PARK/report_router.py:189` | 소유권 검증 없음 |

**대응 방향**:
- 비로그인 세션은 `session_store.py`의 `session_exists()` 로직을 재활용해 요청 session_id가 서버에 존재하는지 확인
- 로그인 유저는 JWT 기반 user_id와 session_id 소유관계 검증 (현재 roadmap_router.py의 voter_id 패턴 참고)
- `integrated_PARK/roadmap_router.py`의 `get_optional_user` + `session_exists` 패턴이 선례

#### 2-2. PII 저장 — HIGH

사용자 질문 원문이 `conversationContext`로 Cosmos DB에 영구 저장됨.

| 파일 | 줄 | 내용 |
|------|----|------|
| `frontend/src/components/ResponseCard.jsx:146` | — | `conversationContext={question}` 전달 |
| `integrated_PARK/feedback_router.py:107` | — | Cosmos DB에 그대로 저장 |

**대응 방향** (스펙에서도 우려 명시):
- 단기: `conversationContext` 저장 비활성화 (프론트에서 전달 중단 or 백엔드에서 null 처리)
- 중기: 저장 시 PII 마스킹 (전화번호, 이름 패턴 regex 치환)

#### 2-3. 입력값 검증 부족 — MEDIUM

`feedback_router.py`의 Pydantic 모델에 길이/형식 제한 없음.

```python
# feedback_router.py FeedbackRequest 현재
session_id: str           # 길이 무제한
tags: Optional[List[str]] # 개수 무제한
conversation_context: Optional[str]  # 크기 무제한
```

**대응 방향**:
```python
from pydantic import Field
from typing import Literal

session_id: str = Field(..., max_length=255)
agent_type: Literal["admin", "finance", "legal", "location", "chat"]
feedback_type: Literal["positive", "negative"]
tags: list[str] = Field(default=[], max_length=10)
conversation_context: str | None = Field(None, max_length=2000)
```

#### 2-4. Rate Limiting 미설정 — MEDIUM

피드백/이벤트/투표 엔드포인트에 요청 횟수 제한 없음. `slowapi` 도입 검토.

```bash
cd integrated_PARK && .venv/bin/pip install slowapi
```

#### 2-5. API_SECRET_KEY 미설정 시 인증 무효화 — HIGH

`integrated_PARK/auth.py:33-34`: `_API_KEY`가 빈 문자열이면 모든 요청 통과.
배포 파이프라인(`.github/workflows/deploy-backend.yml`)에서 환경변수 필수 설정 확인 필요.
