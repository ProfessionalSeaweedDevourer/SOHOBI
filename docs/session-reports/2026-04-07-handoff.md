## 도메인 라우터가 정상인 이유 (참고)

`domain_router.py`는 키워드 매칭 단계(`_keyword_classify`)에서 성공하면 LLM을 호출하지 않음.  
상권·재무 키워드 매칭 성공 → LLM 없이 라우팅 → 에이전트 실행 단계에서 404 발생하는 구조.  
라우팅이 정상이어도 에이전트 응답이 "미분류"로 보이는 이유가 여기 있음.
