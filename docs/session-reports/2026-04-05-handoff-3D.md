### 계층 3-D 동작 방식

- `GET /api/roadmap/votes?session_id=<sid>`: 8개 기능 전체 투표 수 + 현재 세션 투표 여부 반환 (vote_count 내림차순)
- `POST /api/roadmap/vote` body `{feature_id, session_id}`: Cosmos DB `roadmap_votes` 컨테이너에서 토글 (존재하면 삭제, 없으면 생성)
- Cosmos DB 파티션 키: `/feature_id`, 문서 ID: `vote_{feature_id}_{session_id}`
- 인메모리 폴백: `_votes_fallback: dict[str, set]`
- 프론트: localStorage `sohobi_session_id` 로 session_id 읽기, 없으면 버튼 disabled + 안내 메시지
