# docs/session-reports

Claude Code 세션 리포트 및 인수인계 문서 보관 폴더.

---

## 목적

멀티 세션 환경에서 작업 맥락이 단절되지 않도록, 세션 종료 시 현재 상태를 기록하여 다음 세션이 즉시 복원할 수 있게 한다.

## 인수인계 트리거 조건

다음 중 하나라도 해당되면 인수인계 문서를 생성한다:

- 대화 턴 20회 초과
- 동일 에러 3회 이상 반복
- 작업 범위가 최초 요청에서 3개 이상 파일로 확장

## 파일명 패턴

- 세션 리포트: `MM-DD-session.md` 또는 `YYYY-MM-DD-handoff.md`
- 특정 작업 인수인계: `YYYY-MM-DD-주제-handoff.md`

## 규모

프로젝트 기간 동안 **47건**의 세션 리포트/인수인계 문서가 작성되었다.
