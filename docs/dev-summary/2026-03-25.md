## 기능 추가 (Feature)

### 1. 재무 에이전트 차트 출력 및 current_params 누적 — `a02e262` (#47, PARK-2)

CHANG 브랜치(PR #45)의 재무 차트 구현이 orchestrator 충돌로 머지 불가하여, 별도 `PARK-2` 브랜치에서 올바른 구조로 재구현.

- `finance_simulation_plugin.py` — `_generate_chart()` 추가 (matplotlib 히스토그램, base64 PNG 반환)
- `finance_agent.py` — `is_multi` NameError 수정, `return {"draft", "chart", "updated_params"}`
- `orchestrator.py` — `session_vars`→`current_params` 파라미터 수정, dict 반환 분기 처리
- `api_server.py` — `QueryRequest.current_params` 추가, 응답에 `chart`/`updated_params` 포함

**검증**: 2차 질의에서 1차 파라미터(`revenue`, `initial_investment`) 유지 + 신규 파라미터(`rent`) 반영 확인. 차트 base64 PNG ~17KB 정상 생성.

### 2. 차트 한글 폰트 수정 — `03e2e5b` (#48, PARK-2)

- 우선순위 후보: `Apple SD Gothic Neo` → `Nanum Gothic` → `AppleGothic` → `Malgun Gothic`

## 버그 수정 (Fix)

### Signoff gpt-4.1-mini classic 엔드포인트 전환 — `d37b3b2` (#46)

전날(3/24) 미해결이던 signoff API 400 오류를 근본 해결. Cloud Shell 확인 결과 `gpt-4.1-mini`가 classic 엔드포인트(`ejp-9638-resource.openai.azure.com`)에 배포돼 있음을 확인.
