- `integrated_PARK/session_store.py` — `get_user_id_by_session()` 신규
- `integrated_PARK/auth_router.py` — `get_user_info()` 5분 TTL 캐시
- `integrated_PARK/api_server.py` — `/api/v1/logs` enrichment 확장, `/api/v1/logs/users` 신규
- `frontend/src/pages/LogViewer.jsx` — 사용자 필터 드롭다운, 세션 ID 검색 input
- `frontend/src/components/LogTable.jsx` — 응답자 표시 컬럼, 세션 ID·IP 메타 추가
- `frontend/src/pages/PrivacyPolicy.jsx` — AI 학습 활용 조항 삭제, IP 수집 명시, v1.1 시행일 추가

## 성능 최적화 (Perf)

### 백엔드 asyncio 병렬화 및 DB 인덱스 — (#162)

- `getDongCentroids`: 순차 Kakao API 호출 → `asyncio.gather` 병렬화 (~3,000ms → 470ms)
- `get_logs`: session/user 조회 N+1 루프 → `asyncio.gather` 병렬화
- `searchDong`: `LIKE '%q%'` → prefix `LIKE 'q%'` + B-tree 인덱스 추가
- PostgreSQL 상권 쿼리 관련 인덱스 추가

### get_logs 조기 limit + 동 중심좌표 TTLCache — (#163)

- `/api/v1/logs` 전체 로드 후 필터링 → 조기 limit 적용으로 메모리·응답 개선
- 동 중심좌표 계산 결과 TTL 캐시 적용 (24시간)

### 오케스트레이터 retry 최적화 — (#167)

- Sign-off retry 루프의 불필요한 대기 제거
