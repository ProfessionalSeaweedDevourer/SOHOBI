## 배경: signoff 디버깅 연대기

| 단계 | 시도 | 오류 | 원인 |
|------|------|------|------|
| 1 | gpt-5.4-pro + Responses API | 254초 hang | 추론 모델 실제 프롬프트 처리 > Container Apps 240초 timeout |
| 2 | `reasoning={"effort": "low"}` 추가 | 400 unsupported_value | gpt-5.4-pro는 `low` 미지원 |
| 3 | gpt-5-nano | DeploymentNotFound | ejp-9638-resource에 미배포 |
| 4 | gpt-4.1-mini + AI Foundry | 401 wrong API key | 에이전트용 키가 signoff 엔드포인트로 전달 |
| 5 | API 키 폴백 제거 | 401 incorrect audience | `services.ai.azure.com`은 `cognitiveservices.azure.com` 토큰 거부 |
| 6 | `ai.azure.com/.default` 토큰 | 400 API version not supported | AI Foundry `/openai/v1/` 경로와 SDK 경로 구조 불일치 |
| **→** | **내일 classic 엔드포인트로 전환 예정** | | |
